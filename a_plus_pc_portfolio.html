Build Log

Below is a week-by-week record of my progress, including photos, tasks completed, and lessons learned.

Using the resour

  • Week 1-2:Setting up blog in HTML and creating a GitHub repository- Thank goodness I decided to start learning Python a few weeks ago (´꒳`)♡ because it actually came in handy when I used ChatGPT v5.0 to generate the HTML for this blog. Some might say I took the “lazy” way out, but honestly, I’d argue it’s just using my resources wisely—after all, I don’t have an old-school coder hiding in my closet to double-check my work. Coming from a cybersecurity background, coding isn’t my strongest skill, but I know I need to start somewhere, especially if I want a tangible project that shows the effort I’m putting in. The little bits of Python I’ve been practicing (just 15–20 minutes a day on weekdays) surprisingly helped me recognize the logic in HTML, even though I’d never touched it before. The result? My blog is officially up and running, and I even managed to set up a GitHub repository—though I’ll admit it took a fair amount of troubleshooting before the structure and workflow finally clicked. But now that it’s working, I’m excited to keep building and move on to the next step.
  • Week 3-4: Watched A+ 220-1101 Hardware Overview – learned about system unit components.
  • Week 5: Studied mobile device hardware; began planning for desktop vs laptop differences.
